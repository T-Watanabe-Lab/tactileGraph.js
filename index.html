tactileGraph.js

tactileGraph.js is a canvas like tactile graphic drawing library.

Download this project as a .zip file Download this project as a tar.gz file

概要

tactileGraph.jsはブラウザ上で触図データを生成し、印刷用のPNG画像ファイルやエーデルファイルを出力するためのJavaScriptライブラリです。

デモ

点図ドローイング ― マウスを使って点図を描画します
点図グラフ ― CSVデータを読み込んでグラフを描画します
点図関数グラフ ― 数式を読み込んでグラフを描画します
CUI点図エディター ― 文字入力で点図を描画します
点図トレーサー ― 背景画像を読み込んで点図をトレースします
点字カレンダー ― 今月のカレンダーを点字で表示します

tactileGraph.jsを用いたアプリケーション

将棋局面図作成アプリ (PNG一括変換用) ― KIFファイルを読み込んで局面図を作成します。
チェス局面図作成アプリ（FEN） ― FENを読み込んで局面図を作成します。
点描触知文字 デカポワン・エディター ― デカポワンを描画します。
クラインタイプ・エディター ― クラインタイプや平仮名・カタカナの点描文字を描画します。
